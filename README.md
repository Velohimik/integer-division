## How to run
**Making a jar:**

*Windows*
```bash
$\integer-division> .\make.bat
```

*Linux or MacOS*
```bash
$\integer-division> ./make.sh
```


**Running:**
*Windows*
```bash
$\integer-division> \dist\div.bat 123456 789
```

*Linux or MacOS*
```bash
$\integer-division> /dist/div.sh 123456 789
```
